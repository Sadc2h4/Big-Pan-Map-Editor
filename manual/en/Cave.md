# Cave mode

## Unit list
![MD_Description_11](images/MD_Description_11.png)
- Cave mode lists unit folders under arc.
- Use the search box to filter by unit name.
- Selecting a unit loads its pretty preview，3D model，layout，route，and waterbox data.
- If an embedded pretty image exists，the exe resource image is used first.
- When a 3D model is needed，Hocotate_Toolkit.exe is used to generate cache from arc.szs.
***

## Save
- Save writes the active edit target.
- Save All writes layout，route，and waterbox together.
- When editing through cache，saving attempts to repack arc.szs and texts.szs.
- Back up target folders before saving complex edits.
